# Sistem Persamaan Linear

Website ini membahas konsep Sistem Persamaan Linear (SPL)
meliputi:

- Definisi SPL
- Bentuk matriks dan matriks augmented
- Penyelesaian dengan Operasi Baris Elementer (OBE)
- Implementasi algoritma Eliminasi Gauss dengan Python
- Interpretasi geometri menggunakan GeoGebra

---

## Tujuan Pembelajaran

Setelah mempelajari materi ini, pembaca diharapkan mampu:

1. Memahami konsep dasar sistem persamaan linear
2. Mengubah SPL ke dalam bentuk matriks
3. Menyelesaikan SPL dengan metode Eliminasi Gauss
4. Menginterpretasikan solusi SPL secara geometri
